Bilibili Obsidian Clipper

配置指引

视频教程 · GitHub 仓库地址

  1. 在 Obsidian 社区插件市场安装并启用 Local REST API with MCP。
  2. 在插件设置中勾选 Enable Non-encrypted (HTTP) Server，地址通常为 http://127.0.0.1:27123。
  3. 复制 API Key，回到本页填写“Local REST API 地址”和“Local REST API Key”。

填写完成后建议先点击下方“测试连接”。

笔记属性（Frontmatter）

勾选需要写入到 Obsidian 属性区的字段。

自定义属性

支持默认属性的变量映射，比如 `{{upload_date}}`、`{{created}}`

无字幕轨转写

无字幕轨时，插件会临时获取音频并提交到你配置的 Whisper-compatible 服务。FFmpeg 预处理为可选外部服务。

开启后，本地 Whisper 服务会把音频保存到该目录；可在弹窗中删除对应音频。

字幕缓存管理

这里清理的是 Chrome 扩展本地保存的字幕文本缓存，不会删除 Obsidian 笔记、API Key 或本地音频文件。
